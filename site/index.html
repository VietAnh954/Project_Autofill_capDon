    Auto-fill Cấp Đơn¶

    Tự động hoá nhập liệu phiếu cấp đơn bảo hiểm từ Outlook vào file Excel tổng.

    Tại sao có project này?¶

    Hàng ngày, mỗi sale gửi email kèm file Excel/CSV chứa phiếu cấp đơn bảo hiểm (du lịch, sức khỏe, ô tô, xe máy, BHYT/BHXH, HSSV, …). Bạn phải tự mở từng file, copy-paste từng dòng vào file Excel tổng — dễ sai sót, tốn 1-2 giờ/ngày.

    Auto-fill Cấp Đơn giải quyết:

    • Tự đọc mail mới từ Outlook desktop.
    • Tải attachment vào folder local.
    • Phân loại sản phẩm (du lịch / sức khỏe / xe máy…) theo sender + subject + cột.
    • Chuẩn hoá dữ liệu (ngày, CCCD, số điện thoại, tiền tệ).
    • Ghi vào sheet đúng trong file Excel tổng — append-only, có dedup.
    • Backup tự động trước khi sửa.
    • Log từng record, report cuối ngày.

    Demo nhanh¶

    1. Mở Outlook → đọc 20 mail của 8 sale.
    2. Tải 20 file Excel khác nhau.
    3. Mở file tổng, scroll tới sheet đúng.
    4. Copy-paste 100+ dòng.
    5. Kiểm tra trùng lặp tay.
    6. Lưu file, backup.

    ~90 phút/ngày. Hay sai CCCD, ngày tháng.

    python -m auto_fill run

    ~2 phút. Sai sót giảm gần 0.
